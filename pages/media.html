Media Tags




Media tags are tags that display something, like an image, video or audio

for now, we will just talk about how to display an image in html


luckily, HTML has a tag called "<img>"

the "<img>" tag simply lets us display an image


Example



This is the code for the example:


<img src="flag.png" width=200>


This image is the flag of cambodia, this image is originally big but luckily HTML gives us the option to change the width and height.
