Headers

Main Page Cheat Sheet


Headers are used like sub-titles or titles for an HTML page.

It is used to mark important titles or text, like you would want in a website about yourself.


Example


I'm h1, the biggest header

I'm h6, the smallest header

This is the code for the example:


<h1>I'm h1, the biggest header</h1>
<h6>I'm h6, the smallest header</h6>

Note that you don't need to add any bolding to h1, it is automatically bolded.
